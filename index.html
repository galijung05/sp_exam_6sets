세트마다 업종·데이터·출제 함수가 다릅니다. 세트 1~6은 핵심 패턴, 세트 7~9는 변형 함수 조합입니다.

SP 함수 완전 정복

시험에 나오는 함수, 전부 여기 있습니다

IF조건 분기
조건이 참이면 값1, 거짓이면 값2를 반환합니다. 가장 기본적인 분기 함수.
=IF(조건, 참일때값, 거짓일때값)
예시
=IF(D5=1,"남성","여성")
D5가 1이면 "남성", 아니면 "여성" 반환
⚠️ 문제에서 IF 지정 시 CHOOSE·IFS 사용하면 0점
IF 중첩다중 분기
IF 안에 IF를 넣어 3개 이상 조건을 처리합니다. 단가가 3종류일 때 사용.
=IF(조건1, 값1, IF(조건2, 값2, 값3))
예시
=IF(B5="라면",F5*5500,IF(B5="과자",F5*6500,F5*7700))
라면이면 ×5500, 과자면 ×6500, 나머지(음료수)면 ×7700
⚠️ 마지막 항목은 조건 없이 값만 — 괄호 개수 주의!
IFS다중 조건
여러 조건을 나열해서 첫 번째로 참인 조건의 값을 반환합니다. IF 중첩보다 가독성 좋음.
=IFS(조건1, 값1, 조건2, 값2, ...)
예시
=IFS(D5=1,"남성",D5=2,"여성")
D5=1이면 "남성", D5=2이면 "여성"
⚠️ 문제에서 IFS 지정 시 IF로 작성하면 0점
CHOOSE인덱스 선택
인덱스 번호에 해당하는 값을 순서대로 반환합니다. 코드가 1,2,3 연속일 때 깔끔.
=CHOOSE(인덱스, 값1, 값2, 값3, ...)
예시
=CHOOSE(D5,"남성","여성")
D5=1이면 "남성"(1번째), D5=2이면 "여성"(2번째)
⚠️ 인덱스가 1부터 시작. 0이나 범위 초과 시 오류
비교IF vs IFS vs CHOOSE — 언제 어떤 걸 쓸까
함수적합한 상황주의점
IF조건 2~3개, 가장 범용적중첩 깊어지면 괄호 실수 多
IFS조건 3개 이상, 가독성 중요Excel 2019 이상에서만 지원
CHOOSE코드가 1,2,3... 정수 연속일 때코드가 숫자 아니면 사용 불가
실제 시험에서는 문제에 함수가 지정되어 있으면 반드시 그것만 사용해야 합니다. 지정이 없으면 셋 중 아무거나 써도 됩니다.
SUMIF조건 합계
조건 범위에서 조건을 만족하는 행의 합산 범위를 더합니다. 조건 1개.
=SUMIF(조건범위, 조건, 합산범위)
숫자 조건
=SUMIF(F5:F16,"<20",I5:I16)
F열이 20 미만인 행의 I열 합계
텍스트 조건
=SUMIF(H5:H16,"여성",I5:I16)
H열이 "여성"인 행의 I열 합계
⚠️ 문제에서 SUMIF/SUMIFS 지정 시 다른 함수 사용하면 즉시 0점
SUMIFS다중 조건 합계
조건이 2개 이상일 때 사용합니다. 합산범위가 맨 앞에 오는 것이 SUMIF와 다릅니다.
=SUMIFS(합산범위, 조건범위1, 조건1, 조건범위2, 조건2)
예시 — 범위 조건 2개
=SUMIFS(K5:K16,J5:J16,">=10000",J5:J16,"<20000")
J열이 10000 이상 20000 미만인 행의 K열 합계
⚠️ SUMIF와 인수 순서 다름! SUMIFS는 합산범위가 첫 번째
AVERAGEIF조건 평균
조건을 만족하는 행만 골라 평균을 구합니다. SUMIF의 평균 버전.
=AVERAGEIF(조건범위, 조건, 평균범위)
예시
=ROUND(AVERAGEIF(H5:H16,"여성",K5:K16),0)
여성인 행의 K열 계산금액 평균 (반올림)
SUMIF vs SUMIFS헷갈리는 포인트
SUMIFSUMIFS
조건 수1개2개 이상
합산범위 위치세 번째첫 번째
구조(조건범위, 조건, 합산)(합산, 조건범위, 조건, ...)
조건 1개여도 SUMIFS 써도 됩니다. 반대로 조건 2개에 SUMIF는 불가.
COUNTIF조건 개수
조건을 만족하는 셀의 개수를 셉니다. 와일드카드(*,?)로 문자 포함 검색 가능.
=COUNTIF(범위, 조건)
문자 포함 검색
=COUNTIF(C5:C16,"*영*")&"명"
C열에 "영"이 포함된 셀 개수 → "3명" 형태 출력
⚠️ 와일드카드: * = 임의 문자열, ? = 문자 1개. 반드시 큰따옴표 안에!
COUNTIFS다중 조건 개수
조건이 2개 이상일 때 사용. 모든 조건을 동시에 만족하는 행만 셉니다.
=COUNTIFS(범위1, 조건1, 범위2, 조건2)
예시 — 이름+성별 동시 조건
=COUNTIFS(C5:C16,"*영*",H5:H16,"여성")&"명"
이름에 "영" 포함 + 여성인 사람 수
&"단위" 연결텍스트 결합
숫자 결과에 텍스트 단위를 붙여서 출력할 때 사용합니다.
=수식&"단위"
예시들
=COUNTIF(C5:C16,"*영*")&"명" → 3명
=AVERAGEIF(...)&"원" → 125000원
=SMALL(K5:K16,2)&"원" → 45000원
⚠️ &"명" 연결 시 결과가 텍스트가 됨 → 숫자로 다시 계산 불가. 용도에 맞게 사용.
와일드카드패턴 검색
기호의미예시
*임의 개수 문자"*영*" = 어디든 "영" 포함
?문자 정확히 1개"영?" = "영" + 1글자
"영*""영"으로 시작영수, 영희, 영...
"*영""영"으로 끝남수영, 지영, ...영
!VLOOKUP — 단가 테이블 참조
실제 시험에서 단가를 IF 중첩 대신 별도 테이블로 만들어 VLOOKUP으로 참조하는 유형이 출제됩니다.

기본 구조
VLOOKUP수직 검색
찾을 값을 테이블 첫 번째 열에서 찾아 지정한 열 번호의 값을 반환합니다.
=VLOOKUP(찾을값, 테이블범위, 열번호, [일치방식])
단가 테이블 참조 예시
단가표 (N4:O6):
N4: 라면 / O4: 5500
N5: 과자 / O5: 6500
N6: 음료수 / O6: 7700

=VLOOKUP(B5,$N$4:$O$6,2,0)*F5
B5 상품명으로 단가를 찾아 F5 수량 곱하기
⚠️ 핵심 주의사항:
1. 테이블 범위는 반드시 절대참조($) — 아래로 복사해도 범위 고정
2. 열번호: 테이블에서 몇 번째 열인지 (단가는 2번째)
3. 일치방식: 0 = 정확히 일치 (항상 0 사용)
절대참조 ($)필수 개념
수식을 복사해도 범위가 바뀌지 않도록 고정하는 기호.
$열$행 → 행·열 모두 고정
$열행 → 열만 고정
열$행 → 행만 고정
비교
N4:O6 → 복사하면 범위 이동 (오류)
$N$4:$O$6 → 복사해도 범위 고정 (정답)
⚠️ VLOOKUP, RANK 등 범위를 고정해야 하는 함수에서 $ 빠뜨리면 아래 행 복사 시 오류
AVERAGE평균
범위의 평균을 구합니다. 항상 ROUND와 함께 사용해 정수로 반올림.
=ROUND(AVERAGE(범위), 소수자릿수)
예시
=ROUND(AVERAGE(I5:I16),0)
I5:I16 평균을 소수점 첫째자리에서 반올림 → 정수
RANK순위
지정한 값이 범위 내에서 몇 위인지 반환합니다.
=RANK(값, 범위, [순서])
예시
=RANK(K5,$K$5:$K$16,0)
K5가 K5:K16 중 몇 위? (0=내림차순, 1=오름차순)
⚠️ 범위를 절대참조($)로 고정 필수. 아니면 아래 행 복사 시 범위가 밀림
MAX / MIN최대·최소
범위에서 가장 큰 값(MAX) 또는 가장 작은 값(MIN)을 반환합니다.
=MAX(범위) / =MIN(범위)
예시
=MAX(K5:K16) → 계산금액 최댓값
=MIN(K5:K16) → 계산금액 최솟값
LARGE / SMALLN번째 크기
범위에서 N번째로 크거나(LARGE) 작은(SMALL) 값을 반환합니다.
=LARGE(범위, N) / =SMALL(범위, N)
예시
=LARGE(K5:K16,3) → 3번째로 큰 값
=SMALL(K5:K16,2)&"원" → 2번째로 작은 값+"원"
⚠️ MAX = LARGE(범위,1), MIN = SMALL(범위,1)과 같습니다
LEFT왼쪽 추출
텍스트의 왼쪽에서 지정한 글자수만큼 추출합니다.
=LEFT(텍스트, 글자수)
예시
=LEFT(D5,1)
"A-001" → "A" (왼쪽 1글자)
RIGHT오른쪽 추출
텍스트의 오른쪽에서 지정한 글자수만큼 추출합니다.
=RIGHT(텍스트, 글자수)
예시
=RIGHT(D5,3)
"A-001" → "001" (오른쪽 3글자)
MID중간 추출
텍스트의 지정 위치에서 지정 글자수만큼 추출합니다.
=MID(텍스트, 시작위치, 글자수)
예시
=MID(D5,3,3)
"A-001" → "001" (3번째부터 3글자)
ROUND 계열반올림·올림·버림
수치를 지정 자릿수로 처리합니다. 시험에서는 ROUND(정수)가 거의 필수.
=ROUND(수, 자릿수) ← 반올림 =ROUNDUP(수, 자릿수) ← 올림 =ROUNDDOWN(수, 자릿수) ← 버림
자릿수 기준
=ROUND(값, 0) → 정수 (소수점 첫째자리 반올림)
=ROUND(값, -3) → 천단위 반올림
⚠️ 문제에서 "소수점 첫째자리에서 반올림하여 정수로" → ROUND(수식, 0)
1정렬(SORT) 순서 — 실격 주의 항목
정렬은 함수식 기재(⑦⑨⑪) 전에 반드시 먼저 실행해야 합니다.
정렬 후 행 순서가 바뀌면 함수식의 셀 참조도 달라지기 때문입니다.

정렬 실행 방법 (Excel)
① 데이터 범위(B5:K16) 선택
② 데이터 탭 → 정렬
③ 기준 추가: 1순위 H열(성별) → 사용자 지정 목록에서 "여성, 남성" 순서 지정
④ 기준 추가: 2순위 K열(계산금액) → 내림차순
2셀 서식 — 금액 표시
  • 금액 셀(I·J·K열): 통화 형식, ₩ 기호, 천단위 콤마, 소수점 0자리
  • 숫자 셀(수량·연령 등): 콤마 없음, 우측 정렬
  • 문자 셀(성명·성별 등): 가운데 정렬
  • 제목 글꼴 크기: 20pt
  • 할인율(G열)에 ₩ 붙이지 않기 — 백분율 형식 그대로
  • 수량에 콤마 붙이지 않기
3함수식 기재 (⑦⑨⑪) — 텍스트로 보이게
함수식이 계산되지 않고 수식 그대로 출력되어야 합니다.

방법 1: 셀 서식을 텍스트로 변경 후 수식 입력
방법 2: 셀에 '(작은따옴표) 먼저 입력 후 수식 입력
예: '=SUMIF(F5:F16,"<20",I5:I16)
!SUMIF vs SUMIFS 인수 순서
  • =SUMIF(합산범위, 조건범위, 조건) — 틀림!
  • =SUMIF(조건범위, 조건, 합산범위) — 맞음
  • =SUMIFS(합산범위, 조건범위, 조건) — 맞음
!RANK 절대참조 빠뜨리기
  • =RANK(K5,K5:K16,0) → 아래로 복사하면 범위가 K6:K17로 밀림
  • =RANK(K5,$K$5:$K$16,0) → 범위 고정
!와일드카드 따옴표 밖에 쓰기
  • =COUNTIF(C5:C16,*영*) → 오류
  • =COUNTIF(C5:C16,"*영*") → 맞음
!정렬 전에 함수식 기재
  • 수식 먼저 작성 → 정렬 → 함수식 기재 (⑦⑨⑪)
  • 수식 작성 → 정렬 먼저 → 그 다음 함수식 기재
!IF 중첩 괄호 개수 오류
  • =IF(B5="라면",F5*5500,IF(B5="과자",F5*6500,F5*7700) → 괄호 1개 부족
  • =IF(B5="라면",F5*5500,IF(B5="과자",F5*6500,F5*7700))
  • IF 중첩 n번 → 닫는 괄호 n+1개 필요
!SUMIFS 수치에 콤마 넣기
  • ">=${10,000}" → 오류 (콤마가 인수 구분자로 인식)
  • ">=10000" → 맞음 (콤마 없이)
  • 문제에서도 "수치는 10000과 20000을 사용"이라고 명시
실전 작업 & 출력 가이드

Excel 실제 작업 → 출력까지

수식 암기 후 실제 Excel에서 이 순서대로 작업하세요. 출력 주의사항을 지키지 않으면 감점입니다.

1
데이터 입력 (B5:G16)
음영 처리된 행/열 기준(4행, B~G열 헤더)은 입력하지 않습니다. 데이터만 B5부터 입력. 할인율은 5% 형태로 입력하면 자동으로 0.05 처리됩니다.
2
① 성별 수식 — H5에 입력 후 H16까지 채우기
=IF(D5=1,"남성","여성")
H5 입력 → H5 선택 → Ctrl+C → H6:H16 선택 → Ctrl+V
3
② 주문금액 — I5에 입력 후 I16까지 채우기
=IF(B5="라면",F5*5500,IF(B5="과자",F5*6500,F5*7700))
4
③④ 할인금액(J5) · 계산금액(K5) 입력 후 16행까지 채우기
J5: =ROUND(I5*G5,0) K5: =I5-J5
5
정렬 먼저 실행 (수식 기재 전 필수!)
⚠️ 함수식 기재(⑦⑨⑪)는 반드시 정렬 후에 작성합니다. 정렬 전 기재하면 잘못된 셀 참조가 됩니다.
데이터 선택(B5:K16) → 데이터 탭 → 정렬 → 기준1: H열(성별) 사용자 지정 순서(여성→남성) → 기준2: K열(계산금액) 내림차순
6
⑤ 평균 (17행) · ⑥⑦⑧⑨⑩⑪ (18~23행)
I17: =ROUND(AVERAGE(I5:I16),0) K18: =SUMIF(F5:F16,"<20",I5:I16) K20: =COUNTIF(C5:C16,"*영*")&"명" K22: =SUMIFS(K5:K16,J5:J16,">=10000",J5:J16,"<20000")
⑦⑨⑪(함수식 기재 셀): 해당 셀 서식을 텍스트로 변경 후 수식 그대로 타이핑. 앞에 '(작은따옴표) 붙여도 됨.
7
셀 서식 적용
금액 셀(I·J·K열): 통화 형식, ₩ 표시, 소수점 0자리, 천단위 콤마
숫자 셀 우측 정렬 / 문자 셀 가운데 정렬
제목(상품별 주문 금액 계산): 글꼴 크기 20pt
1
데이터 범위 선택 — 여성 행만!
⚠️ 실격 사유: 전체 데이터를 선택하면 즉시 실격. 정렬 후 여성 행(C열 성명 + I열 주문금액 + K열 계산금액)만 Ctrl로 비연속 선택.
예) 정렬 후 여성이 5행~10행이면 → C5:C10, I5:I10, K5:K10을 Ctrl 누르고 선택
2
차트 삽입 → 혼합형 선택
삽입 탭 → 차트 → 콤보(혼합형) 선택
주문금액: 묶은 세로 막대형
계산금액: 꺾은선형(표식 있음)
3
차트 요소 설정
제목: 여성 주문 금액 계산
X축 제목: 성명 / Y축 제목: 금액
범례: 계산금액, 주문금액 순서
계산금액 계열만 → 데이터 레이블 추가 → 위쪽
4
위치·크기 조정
작업표 하단에 배치. 그래프 너비 = 작업표 너비에 맞춤. 작업표+그래프가 A4 1장에 들어가도록 인쇄 미리보기로 확인.
1
피벗테이블 삽입
범위 B4:K16 선택 → 삽입 탭 → 피벗테이블 → 새 워크시트
2
필드 배치
행 영역: 할인율 → 성명 순서로 드래그
값 영역: 할인금액 합계, 계산금액 합계
값 필드 설정 → 표시 형식 → 숫자 → 천단위 콤마, 소수점 0
3
그룹 추가
할인율 2% 셀 우클릭 → 그룹 → "비회원" 이름 입력
5%·7% Ctrl 선택 → 우클릭 → 그룹 → "회원"
10% 선택 → 우클릭 → 그룹 → "우수회원"
그룹 필드명을 "회원 구분"으로 변경
4
조정1 · 조정2 출력
조정1: 회원 구분 기준 — 비회원·우수회원 확장, 회원 축소(+ 클릭해서 접기)
조정2: 할인율 기준 — 2% 확장, 5%·7%·10% 축소
각각 A4 1/2장 크기로 상단·하단 배치. 제목 텍스트 박스로 추가.
!
상단 여백 6cm 설정 — 모든 출력물 공통
⚠️ 모든 출력물(SP·DB·PT) 상단 여백을 반드시 약 6cm(60mm)로 맞춰야 합니다. 이 공간에 자필로 비번호/수험번호/성명을 기재합니다.
페이지 레이아웃 탭 → 여백 → 사용자 지정 여백 → 위쪽: 6cm
1
인쇄 방향: 세로(좁게)
페이지 레이아웃 → 용지 방향 → 세로. A4 기준.
2
인쇄 미리보기로 반드시 확인
Ctrl+P → 미리보기에서 확인사항:
• 작업표 + 그래프가 1장에 다 들어오는지
• 셀 내용이 잘리지 않는지 (열 너비 조정)
• ₩ 기호와 숫자가 제대로 표시되는지
• 피벗테이블이 판독 가능한지
3
출력 순서 & 자필 기재
출력 후 각 장 중앙 상단(6cm 여백 안)에 자필로:
비번호 / 수험번호 / 성명
각 장 중앙 하단에 자필로 쪽번호

편철 순서: ①답안표지 → ②SP작업표+그래프 → ③SP피벗 → ④DB조회화면 → ⑤DB보고서 → ⑥PT슬라이드
4
출력은 1회 원칙
출력 작업은 단 1회. 화면과 출력물이 다를 경우에만 1회 추가 가능. 기계 결함 시 예외.
출력 전 파일 저장 필수 (Ctrl+S). 파일 내용과 출력물이 다르면 즉시 실격.
